About this project

The problem

In our daily lives, we often tend to throw away food that have been left over from the previous days, without noticing the sort of consequences that it could have for our environment. Well, it's quite simple, though, isn't it? We tend to not see something as dirty or polluting anymore when it's out of our sign.

However, if we look at it from a global perspective, there's billions of households throwing away leftovers that keep stacking up in landfills; the damage to the land is massive, and lots of money is wasted on both food and expanding the landfills.

My solution

As a result, I decided to create Replate, a simple yet solid solution as a way to cut down food waste at an individual level. By allowing users to scan their food via camera, or input the name as text, it sends the information to ChatGPT and lets them get suggestions on what to do with their leftover food, rather than spend time search various ways for each food item.

Technical information

Tools used

  • Web application: Django Framework with Python
  • Webpages: HTML, CSS, Javascript
  • Image detection: Clarifai API
  • Sending answers: ChatGPI API
  • Drawing UI Design: Figma

How this project works

  1. First, it asks the user to input a picture of food, or enter its name as text.
  2. If it's an image, then the data is sent to Clarifai API to detect what food it contains. If it's text, then the text is uploaded directly.
  3. The data for the food is then sent to ChatGPT's API to ask for suggestions on how to reuse the leftover food.
  4. The suggestions from ChatGPT is output back to the user's browser.
